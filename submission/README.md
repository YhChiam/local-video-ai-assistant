# Submission Assets

This folder contains assignment submission artifacts.

## Expected layout

- input-videos/: one or more example `.mp4` files used for testing
- generated-outputs/pdf/: generated PDF report samples
- generated-outputs/pptx/: generated PowerPoint samples
- docs/: optional screenshots or supporting notes
- checklist/: final handoff notes

## What to include before submission

1. At least one short input video in `input-videos/`.
2. At least one generated PDF report in `generated-outputs/pdf/`.
3. At least one generated PPTX deck in `generated-outputs/pptx/`.
4. A short final summary in `checklist/final-summary.md`.
